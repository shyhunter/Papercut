3. **Synced scrolling:**
   - Add `useRef<HTMLDivElement>` for both Before and After scroll containers.
   - On `onScroll` of either panel, read `scrollTop` and `scrollLeft` and set them on the other panel.
   - Use a `isSyncing` ref flag to prevent infinite scroll loops:
     ```
     const isSyncing = useRef(false);
     const beforeScrollRef = useRef<HTMLDivElement>(null);
     const afterScrollRef = useRef<HTMLDivElement>(null);

     function handleScroll(source: 'before' | 'after') {
       if (isSyncing.current) return;
       isSyncing.current = true;
       const src = source === 'before' ? beforeScrollRef.current : afterScrollRef.current;
       const tgt = source === 'before' ? afterScrollRef.current : beforeScrollRef.current;
       if (src && tgt) {
         tgt.scrollTop = src.scrollTop;
         tgt.scrollLeft = src.scrollLeft;
       }
       requestAnimationFrame(() => { isSyncing.current = false; });
     }
     ```
   - Attach `ref` and `onScroll` to each panel's scrollable `<div>` (the `overflow-auto` div inside PreviewPanel).
   - Update `PreviewPanel` to accept `scrollRef` and `onScroll` props, or refactor to pass refs directly.

4. **Rename "Process another" to "Start Over":**
   - Change the text from "Process another" to "Start Over" in the bottom strip button.
   - Keep data-testid as "process-another-btn" (E2E tests reference it).

5. **Simplify bottom strip:**
