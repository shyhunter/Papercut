**src/lib/pdfSplit.ts** — Split engine using pdf-lib:

Types:
```ts
type SplitMode =
  | { type: 'ranges'; ranges: PageRange[] }       // e.g., [{start:1,end:3}, {start:5,end:5}, {start:7,end:10}]
  | { type: 'every-n'; n: number }                 // split every N pages
  | { type: 'individual' };                        // each page separate

interface PageRange {
  start: number;  // 1-based
  end: number;    // 1-based, inclusive
}

interface SplitOutput {
  fileName: string;       // e.g., "document_pages_1-3.pdf"
  bytes: Uint8Array;
  pageCount: number;
  pageNumbers: number[];  // 1-based page numbers included
}

interface SplitResult {
  outputs: SplitOutput[];
  sourceFileName: string;
  sourceTotalPages: number;
}
```

`splitPdf(pdfBytes: Uint8Array, sourceFileName: string, mode: SplitMode): Promise<SplitResult>`:
1. Load source PDF with `PDFDocument.load(pdfBytes)`
2. Based on mode:
   - **ranges**: For each range, create new PDFDocument, copyPages for the range, save
   - **every-n**: Split into chunks of N pages, each becomes a separate output
   - **individual**: Each page becomes its own output
3. Auto-generate file names from source name + page numbers (e.g., "report_pages_1-3.pdf", "report_page_5.pdf")
4. Save each output with `useObjectStreams: true`
5. Return array of SplitOutput with bytes, fileName, pageCount, pageNumbers

Also export `parsePageRangeText(text: string, totalPages: number): PageRange[]`:
- Parse text like "1-3, 5, 7-10" into PageRange array
- Validate: ranges within bounds, no overlaps
- Throw descriptive error on invalid input
  </action>
