  <name>Task 1: PDF text extraction engine</name>
  <files>
    src/lib/pdfTextExtract.ts
  </files>
  <action>
Create `src/lib/pdfTextExtract.ts` for extracting text with position and font data from PDFs:

1. **ExtractedTextItem interface** (exported):
   - id: string (generated UUID for tracking edits)
   - text: string
   - x: number (PDF points from left edge)
   - y: number (PDF points from bottom edge)
   - width: number
   - height: number (derived from font size)
   - fontSize: number
   - fontName: string
   - transform: number[] (the raw 6-element transform matrix for precise positioning)

2. **extractPageText(pdfBytes: Uint8Array, pageIndex: number): Promise&lt;ExtractedTextItem[]&gt;**
   - Load PDF with `pdfjsLib.getDocument({ data: pdfBytes.slice() })` — MUST use .slice() per project memory (StrictMode safety)
   - Get page with `doc.getPage(pageIndex + 1)` (pdfjs is 1-indexed)
   - Call `page.getTextContent()` to get TextContent
   - Filter items to TextItem type (items with 'str' property)
   - Map each TextItem to ExtractedTextItem:
     - Extract transform matrix: `[scaleX, skewY, skewX, scaleY, translateX, translateY]`
     - x = translateX, y = translateY
     - fontSize = Math.abs(scaleY) (font size encoded in transform)
     - fontName = item.fontName
     - width = item.width
     - height = Math.abs(scaleY) (approximate)
     - Generate unique id with crypto.randomUUID()
   - Return sorted by y descending then x ascending (top-to-bottom, left-to-right reading order)

3. **extractAllPagesText(pdfBytes: Uint8Array): Promise&lt;Map&lt;number, ExtractedTextItem[]&gt;&gt;**
   - Load PDF once, iterate pages, extract text for each
   - Return Map keyed by pageIndex (0-based)

4. **getPageDimensions(pdfBytes: Uint8Array, pageIndex: number): Promise&lt;{width: number, height: number}&gt;**
   - Get viewport at scale 1.0
   - Return width and height in PDF points

Use cancelled flag pattern for async operations (project convention for StrictMode safety).
  </action>
