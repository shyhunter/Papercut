<task type="auto">
  <name>Task 1: Document converter orchestrator library</name>
  <files>
    src/lib/documentConverter.ts
  </files>
  <action>
Create `src/lib/documentConverter.ts` that orchestrates document conversion:

1. **convertDocument(sourcePath, sourceFormat, options: ConvertOptions): Promise&lt;ConvertResult&gt;**
   - Determine which engine to use based on output format:
     - DOCUMENT_FORMATS (docx, doc, odt, txt, rtf, pdf) -> LibreOffice
     - EBOOK_FORMATS (epub, mobi, azw3) -> Calibre
   - For LibreOffice: write source bytes to temp file (using Tauri fs), invoke `convert_with_libreoffice` Rust command, receive output bytes
   - For Calibre: write source bytes to temp file, build extra_args from ConvertOptions (font, margins, spacing), invoke `convert_with_calibre`, receive output bytes
   - Return ConvertResult with outputBytes, outputFormat, originalSize, outputSize

2. **buildCalibreArgs(options: ConvertOptions): string[]**
   - Map ConvertOptions to Calibre CLI flags:
     - fontFamily -> `--base-font-size` + `--font-size-mapping`
     - fontSize -> `--base-font-size`
     - margins -> `--margin-top`, `--margin-right`, `--margin-bottom`, `--margin-left`
     - lineSpacing -> `--line-height`
     - epubLayout -> for 'fixed': `--output-profile tablet`; for 'reflowable': default
   - Return array of CLI args

3. **getAvailableOutputFormats(inputFormat: ConvertFormat): ConvertFormat[]**
   - PDF input -> all non-PDF formats (docx, doc, odt, epub, mobi, azw3, txt, rtf)
   - DOCX/DOC/ODT input -> pdf + ebook formats + other doc formats
   - EPUB input -> pdf, mobi, azw3
   - TXT/RTF input -> pdf, docx, epub
   - Never returns the same format as input

4. **checkSidecarAvailability(): Promise&lt;{libreoffice: boolean, calibre: boolean}&gt;**
   - Try invoking `soffice --version` via shell command
   - Try invoking `ebook-convert --version` via shell command
   - Return availability status for UI feedback
   - Cache the result (check once per app session)

5. **getEngineForFormat(format: ConvertFormat): ConverterEngine**
   - Returns which engine handles this format

Use `@tauri-apps/api/core` invoke() for Rust command calls. Handle errors with descriptive messages: "LibreOffice is required for this conversion. Install it from libreoffice.org" / "Calibre is required for ebook conversion. Install it from calibre-ebook.com".
  </action>
