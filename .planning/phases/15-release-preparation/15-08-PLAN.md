       - Paddle (Tauri-friendly, handles tax)
       - Stripe + custom license server (most control)
       - Recommendation: Paddle or Gumroad for simplicity
    6. **License key architecture:**
       - Simple stub approach for now: `isLicensed()` function returns `true`
       - Future: license key validation via payment provider API
       - No pre-wiring needed for beta — add licensing code when payment provider is chosen
    7. **Timeline:** Post-beta phase, after feedback validates demand
