}

1. Create `src/components/pdf-editor/EditorCanvas.tsx`:
   - Renders all PDF pages stacked vertically in a scrollable container
   - Each page rendered to a canvas element using pdfjs-dist (use `pdfBytes.slice()` per StrictMode rule)
   - Gap between pages: 16px with a subtle shadow on each page
   - Background: neutral gray (#e5e5e5 light / #1a1a1a dark)
   - Pages are centered horizontally in the canvas area
   - Uses IntersectionObserver to track which page is most visible → updates `currentPage` in context
   - Only renders pages near the viewport (virtualization): render current page +/- 2 pages, show placeholder divs for others

   - On mount, calculate `fitWidthZoom` = containerWidth / pageWidth and set it as default zoom
