1. Add drag-to-reorder to PagePanelThumbnail:
   - Use HTML5 drag and drop (draggable attribute + onDragStart/onDragOver/onDrop)
   - Drag visual: opacity on dragged thumbnail, blue insertion line between thumbnails on hover
   - When dropped: call `reorderPages(fromIdx, toIdx)` in EditorContext
   - If multiple pages selected, drag moves all selected pages as a group

2. Implement pdf-lib page operations in EditorContext:
   - `addBlankPage(afterIdx)`: Create new PDFDocument, add blank A4 page, copy it into the current doc
   - `addPagesFromPdf(afterIdx, otherPdfBytes)`: Load other PDF with pdf-lib, copy all its pages into current doc at position
   - `deletePages(indices)`: Copy all pages EXCEPT the given indices to a new PDFDocument, replace pdfBytes
   - `duplicatePages(indices)`: For each index, copyPages from current doc to itself
   - `reorderPages(fromIdx, toIdx)`: Rebuild document with pages in new order
   - After each operation: serialize with `doc.save({ useObjectStreams: false })`, update pdfBytes in state, update pageCount, update pages array, re-render thumbnails and canvas

3. Wire "Insert from PDF file" to open a file picker (dialog.open with PDF filter), read the file, pass bytes to `addPagesFromPdf`.

4. Wire Delete and Duplicate buttons to their respective context actions.
