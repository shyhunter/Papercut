    **pdfRedact.ts:**
    Implement true permanent redaction using the render-to-image approach (research recommendation -- safest method, guarantees no extractable text remains).

    ```typescript
    async function applyRedactions(pdfBytes: Uint8Array, redactions: RedactionRect[]): Promise<Uint8Array>
    ```

    Process:
    1. Group redactions by pageIndex
    2. Load PDF with pdfjsLib.getDocument({ data: pdfBytes.slice() })
    3. Create new PDFDocument with pdf-lib: PDFDocument.create()
    4. For each page in the original PDF:
       a. If page has NO redactions: copy the page as-is via newDoc.copyPages(sourceDoc, [pageIndex]) -- preserves text selectability on non-redacted pages
       b. If page HAS redactions:
          - Render page to canvas via pdfjs-dist at scale 2.0 (high quality)
          - Draw black rectangles over redacted areas on the canvas (convert percentage coords to canvas pixel coords)
          - Export canvas to PNG bytes (canvas.toBlob)
          - Create a new blank page in the output doc with same dimensions as original
          - Embed the PNG image and draw it to fill the entire page
          - This GUARANTEES no text data remains on redacted pages (the page IS an image)
    5. Save output doc and return bytes

    Load source doc with pdf-lib too (PDFDocument.load) for copying non-redacted pages.
    Coordinate conversion: rect.x% * canvasWidth / 100, rect.y% * canvasHeight / 100, etc.

    IMPORTANT: Document the tradeoff -- text on redacted pages becomes non-selectable. This is the correct behavior for a redaction tool (security over convenience).

    **RedactPdfFlow.tsx:**
    Complete 3-step flow: Select PDF -> Redact -> Save

    Step 0 (Pick): File picker for PDF (same pattern). Consume pendingFiles. Read file bytes via readFile.

    Step 1 (Redact): Render RedactStep with pdfBytes. When user clicks "Apply Redactions", receive the redactions array. Show processing spinner while applyRedactions runs. On success: store result bytes, advance to step 2.

    Step 2 (Save): Render SaveStep with resultBytes, defaultSaveName "{base}-redacted.pdf".
    - Show a note above SaveStep: "Redacted pages have been flattened to images. Text on those pages is no longer selectable."
    - Show redaction summary: "X redaction(s) applied across Y page(s)"

    Wrap in StepErrorBoundary.
