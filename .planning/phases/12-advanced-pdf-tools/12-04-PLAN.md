<task type="auto">
  <name>Task 1: pdfSign.ts library and SignaturePlaceStep component</name>
  <files>src/lib/pdfSign.ts, src/components/sign-pdf/SignaturePlaceStep.tsx</files>
  <action>
    **pdfSign.ts:**
    Create the signature embedding function following the pdfWatermark.ts pattern.

    ```typescript
    interface SignatureOptions {
      imageBytes: Uint8Array;  // PNG bytes from data URL
      x: number;               // PDF coords (bottom-left origin)
      y: number;
      width: number;
      height: number;
      pageIndices: number[];   // Which pages to stamp
    }

    async function addSignature(pdfBytes: Uint8Array, options: SignatureOptions): Promise<Uint8Array>
    ```

    - Load PDF with PDFDocument.load(pdfBytes, { ignoreEncryption: true })
    - Embed signature image via doc.embedPng(options.imageBytes)
    - For each pageIndex in options.pageIndices: call page.drawImage(sigImage, { x, y, width, height })
    - Save with { useObjectStreams: true } (NOT useCompression per project decision)
    - Return new Uint8Array

    **SignaturePlaceStep.tsx:**
    Props: pdfBytes (Uint8Array), signatureDataUrl (string), onComplete (resultBytes: Uint8Array) => void, onBack () => void

    Layout:
    - Left: Page preview using PagePreview component with interactive overlay
    - Right: Controls panel (page navigation, page range selector, apply button)

    Page navigation:
    - Show current page number and total ("Page 3 of 10")
    - Previous/Next buttons to navigate between pages
    - Page selector dropdown for quick jumps

    Signature overlay (rendered as children of PagePreview):
    - Display the signature image as an absolutely-positioned div
    - Drag-and-drop: mousedown on signature starts drag, mousemove updates position, mouseup ends drag. Track position as { x, y } in screen coordinates relative to the page preview container.
    - Resize handles: 4 corner handles (small squares). Dragging a corner resizes the signature maintaining aspect ratio.
    - Initial placement: center of page, 150px wide (scaled proportionally)

    Page range selector:
    - Three options: "Current page only", "All pages", "Custom range"
    - Custom range: text input accepting "1-3, 5, 7-10" format with validation
    - Parse custom range into page indices array

    Apply button:
    - Convert screen coordinates to PDF coordinates: pdfX = screenX * (pdfWidth / previewWidth), pdfY = pdfHeight - (screenY + sigHeight) * (pdfHeight / previewHeight). CRITICAL: PDF uses bottom-left origin, screen uses top-left (research pitfall 2).
    - Convert signature data URL to Uint8Array (base64 decode from data URL)
    - Call addSignature with computed options
    - Show processing spinner
    - On success: call onComplete with result bytes
  </action>
  <verify>Run `npx tsc --noEmit`. Test signature placement visually -- drag, resize, verify position mapping to PDF coordinates.</verify>
  <done>Signature can be dragged onto page preview, resized via corners, and applied to selected page range. PDF output has signature at correct position.</done>
